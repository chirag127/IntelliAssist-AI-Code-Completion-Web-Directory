## üöÄ Features

- **Curated Listings:** Hand-picked selection of the top 50+ AI coding assistants.
- **Interactive Filtering:** Dynamic filtering based on integration type, language, and more.
- **Comparative Metrics:** Side-by-side view of key product attributes.
- **Frontend-Only Architecture:** No backend, no servers, no databases. Runs entirely in your browser.
- **User-Provided API Keys:** Securely use the application with your own API keys.
- **Modern UI:** A sleek, responsive, and "Spatial-Adaptive" user interface built with Tailwind CSS.
